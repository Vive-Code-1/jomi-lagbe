

## LandDetail পেজ Heritage Modernist ডিজাইনে আপডেট

### যা করা হবে

রেফারেন্স ইমেজ অনুসারে জমি বিস্তারিত পেজটি পুরোপুরি রিডিজাইন করা হবে। লেআউটে বড় ইমেজ + ডানপাশে মালিকের তথ্য কার্ড, নিচে বিস্তারিত বর্ণনা ও জমির প্রপার্টি গ্রিড থাকবে।

### লেআউট (রেফারেন্স ইমেজ অনুসারে)

```text
┌─────────────────────────────┬──────────────┐
│  বড় ইমেজ (থাম্বনেইল সহ)    │ মালিকের তথ্য  │
│                             │ (নাম, ফোন)   │
│                             │              │
│                             │ মালিকের নম্বর  │

│ বিস্তারিত বর্ণনা              │ কোয়ালিটি     │

├──────┬──────┬──────┬────────┤              │
│বয়স   │সাইজ  │রোড   │দাম/শতক │              │

```

### ধাপসমূহ

1. **LandDetail.tsx পুরোপুরি রিডিজাইন**
   - Heritage Modernist ডার্ক-লাইট কম্বো স্টাইল
   - বড় ইমেজ সেকশন (ইমেজ গ্যালারি থাম্বনেইল সহ)
   - ডানপাশে মালিকের তথ্য কার্ড (সবুজ বর্ডার, আইকন সহ)
   - "মালিকের নম্বর আনলক" বাটন (সবুজ primary)
   - বিস্তারিত বর্ণনা সেকশন (ধূসর ব্যাকগ্রাউন্ড)
   - প্রপার্টি ইনফো গ্রিড: বয়স, সাইজ (শতক), রাস্তার প্রস্থ, দাম/শতক, ইত্যাদি
   - "মানচিত্র অবস্থান" সেকশন (প্লেসহোল্ডার)
   - টিপস/পুরস্কার সেকশন নিচে

2. **i18n আপডেট** — নতুন কী যোগ (প্রতি শতক দাম, প্রপার্টি বৈশিষ্ট্য লেবেল ইত্যাদি)

3. **Footer কম্পোনেন্ট যোগ** — পেজের নিচে

### টেকনিক্যাল ডিটেইলস

- `src/pages/LandDetail.tsx` — সম্পূর্ণ রিরাইট
- `src/lib/i18n.tsx` — নতুন ট্রান্সলেশন কী যোগ
- বিদ্যমান Supabase কোয়েরি ও ফেভারিট/আনলক লজিক অপরিবর্তিত থাকবে
- Heritage Modernist কাস্টম CSS টোকেন ব্যবহার (surface, primary, outline ইত্যাদি)
- প্রতি শতক দাম ক্যালকুলেশন: `land.price / land.area_size`
